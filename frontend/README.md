# MyChamber Frontend

Premium React frontend for Doctor Chamber & Patient Appointment Management.

## Stack
- React + Vite
- Tailwind CSS
- React Router DOM
- Axios
- Framer Motion
- Recharts
- Lucide React
- React Hot Toast

## Run Locally
1. Start backend on `http://localhost:5000`
2. In this folder:

```bash
npm install
npm run dev
```

Frontend runs on `http://localhost:5173`.

## API Base URL
Configured in `src/services/api.js`:
- `http://localhost:5000/api`
